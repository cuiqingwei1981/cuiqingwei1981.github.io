3. brew

升级更新 brew update cd /usr/local git reset –hard git checkout . sudo chown -R cuiqingwei /usr/local brew update http://www.xitongzhijia.net/xtjc/20141204/31879.html

4. MacPorts

https://www.macports.org/install.php http://guide.macports.org/#installing.macports

5. wget

sudo port install wget

6. zsh

wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh http://leeiio.me/bash-to-zsh-for-mac/ http://www.sketchup.com/download/all 打开 “终端” 输入： chflags nohidden ~/library/ 即可显示被隐藏的 Library（资源库）!

7. mac say 唱歌

应用程序>实用工具>Terminal 复制和黏贴以下内容，然后回车：

  • spotlight搜索框中输入”终端”或者”terminal”, 找到并打开”终端”这个程序。
  • 在下面引用框里单击三下，cmd+c复制歌词，cmd+v粘贴到终端窗口，回车。
  • 开始听歌了。
